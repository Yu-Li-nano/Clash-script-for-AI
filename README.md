# Clash Verge Global Extension Script

This repository stores a Clash Verge global extension script.

## File

- `clash-verge-global-extend.js`: filters unwanted proxy nodes, creates an `AI代理` proxy group, and prepends custom direct/proxy routing rules.

## Usage

Paste the contents of `clash-verge-global-extend.js` into Clash Verge's global extension script editor.
